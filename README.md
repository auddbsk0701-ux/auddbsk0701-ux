### CAF-AD: 클래스 인지 기반 산업 이미지 이상 탐지 모델
**스마트팩토리 캡스톤디자인 | 2026**

다양한 제품을 하나의 모델로 검사하는 통합 이상 탐지 환경에서,  
제품별 특성을 반영할 수 있는 클래스 인지형 비전 검사 모델을 개발했습니다.

- 기존 통합 이상 탐지 모델인 Dinomaly의 구조와 한계를 분석하고 개선 방향 도출
- 제품 클래스에 따라 모델의 특징 재구성 방식을 조절하는 FiLM 기반 모듈 구현
- 소수의 결함 데이터를 활용해 탐지 성능을 보완하는 Few-shot 학습 방식 적용
- Real-IAD와 MVTec-AD 데이터셋을 활용한 모델 학습 및 성능 비교
- GPU 학습 환경 구성과 클래스별 결과 분석 및 이상 영역 시각화 수행
- Streamlit을 활용해 이미지 입력, 이상 점수 확인 및 결함 영역 시각화가 가능한 데모 구현
- 기존 모델 대비 이미지 수준 이상 탐지 성능을 개선하고, 성능이 낮았던 제품군에서 향상 효과 확인

**담당 역할:** FiLM 모듈 구현, GPU 환경 설정, 실험 결과 분석 및 시각화, 데모 구현  
**활용 기술:** Python, PyTorch, DINOv2, Streamlit, Google Colab
